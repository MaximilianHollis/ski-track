## Ski Track
Crowdedness tracker for ski resorts

#

### Setup

With [cotton](https://cotton.js.org/)

```bash
cotton install
cotton run dev
```

With yarn

```bash
yarn
yarn dev
```

Client (UI): https://github.com/MaximilianHollis/ski-track-client

AI Inference API: https://github.com/MaximilianHollis/ski-track-inference
